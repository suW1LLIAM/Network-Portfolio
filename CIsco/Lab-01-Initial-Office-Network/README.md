# Lab 01 - Initial Office Network

## Scenario

A new company called **Zintan Tech Solutions (ZTS)** has moved into a new office.

As the Junior Network Engineer, my first task is to build the initial local network that allows employees to communicate with each other.

---

## Objective

- Configure a Cisco switch.
- Assign hostnames.
- Configure IP addresses.
- Verify end-to-end connectivity.
- Save the configuration.
